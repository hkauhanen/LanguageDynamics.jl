LanguageDynamics.EmptySpeaker — Type

An empty speaker. This is an empty type that has no functionality apart from 'existing'. Its use case is testing and debugging population-dynamic functions; linguistically, empty speakers are inert. They can be rendezvoused, technically, but nothing happens.
